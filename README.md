# Student Attendance Tracking System

## Project Description

This project contains a simple Django website with two main pages:
 ## Home Page: The main landing page of the website
 ## About Page: Provides information about the websites or organization.

The pages are created using Django Class-Based Views(CBV) with TemplateView


## Features involved
1. Home page for website introduction
2. About page for project information
3. Implemented using Django Class-Based Views
4. Simple URL routing system
5. Clean and organized project structure
6. Responsive HTML templates


## How to Run
1. Clone the project:
    git clone
    <repository-url>
    cd project-name
2. Create and activate virtual environment
3. Install dependencies:
    pip install -r requirements.txt
4. Apply migrations:
    python manage.py migrate
5. Start server:
    python manage.py runserver
